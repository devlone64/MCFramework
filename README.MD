# MCFramework
Framework for minecraft bukkit/proxy plugin. (Bukkit, Bungeecord, etc...)
- Inventory classes.
- Configuration classes.
- Dependencies classes.
- etc classes.

## Supported Platform
Bukkit(Spigot, Paper, etc...), Bungeecord(Coming Soon)

## Supported Version
1.17.X - 1.21.4

## Java Version
jdk 21.0.5
